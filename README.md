- Factorial
- Consecutive sum
- Fibonacci
- GCD

- Upper case conversion
- Lower case conversion
- Capetalization
- Alphabetization

### Array Manipulation

- Print array
- Maximum item
- Minimum item
- Sum
- Product
- Reverse
- Merge
- Remove duplicates

- Sentinel search
- Ordered sequential search
- Binary search

### Sorting Algorithms

- Bubble sort
- Insertion sort
- Shell sort
- Merge sort
- Quick sort
